Gmail / Email Notifications
📧 EmailJS Setup — Get New Student Alerts on Gmail
📋 STEP-BY-STEP SETUP:
Step 1 → Go to emailjs.com → Create free account (200 emails/month free)
Step 2 → Add Email Service → Connect your Gmail account → Copy Service ID
Step 3 → Create Email Template → Use variables below → Copy Template ID
Step 4 → Go to Account → Copy your Public Key
Step 5 → Paste all 3 values below → Click Save → Test it!
📝 Email Template Variables to use in EmailJS:
{{student_name}} — Full name of student
{{student_email}} — Email address
{{student_phone}} — Phone number
{{student_grade}} — Class / Grade
{{student_board}} — Board (CBSE etc)
{{join_date}} — Date they registered
📨 Code to Add to register.html

Add this script tag in the <head> of register.html, then uncomment the emailjs.send() block:

<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>
<script>emailjs.init('YOUR_PUBLIC_KEY');</script>
Razorpay Setup & Bank Connection
🏦 Step-by-Step: Connect Bank Account to Razorpay
1
Create Razorpay Account
Go to dashboard.razorpay.com → Sign Up with your mobile number and email. Use your real name as it will appear on student payments.
2
Complete KYC Verification
Go to Settings → Account & Billing. Upload:
• PAN Card (mandatory)
• Aadhaar Card OR Passport
• Bank account details (account number + IFSC)
• Business proof (for tuition center: your Aadhaar is enough as individual)
3
Add Bank Account
Settings → Bank Account → Add your savings/current account. Razorpay will do a ₹1 penny drop verification. Money settles to your bank in T+2 days (2 business days after payment).
4
Get API Keys
Settings → API Keys → Generate Test Key. You get:
• Test Key ID — use while developing (starts with rzp_test_)
• Live Key ID — use when real students pay (starts with rzp_live_)

Razorpay charges 2% per transaction. On ₹2000 course = ₹40 fee, you get ₹1960.
5
Paste Keys into courses.html
Open courses.html → Find the line:

const RAZORPAY_KEY = 'rzp_test_XXXXXXXXXXXXXXXXXX';

Replace with your actual key. Switch to live key before launch.
